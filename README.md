## Install and Run

```bash
yarn install
yarn test
```

## Majestic

[Nice simple test dashboard](https://github.com/Raathigesh/majestic)

```bash
npx majestic
```

or install globally

```bash
yarn global add majestic
```

### http-server for node, needed to run the local example response file

```bash
npm install --global http-server

http-server .
```
